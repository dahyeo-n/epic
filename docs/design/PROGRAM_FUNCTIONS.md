# Epic 기능 명세

[인덱스로 돌아가기](../README.md)

## 프로젝트 개요
Epic은 "서하시/대하 드라마, 삶의 기록을 거대한 이야기로 만든다"는 컨셉의 블로그 기반 웹 애플리케이션입니다.

## 기본 기능

### 1. 사용자 인증 (Authentication & Authorization)
- **회원가입**
  - 사용자 계정 생성
  - 아이디(이메일) 중복 확인
  - 비밀번호 암호화 저장
  
- **로그인/로그아웃**
  - 세션 기반 인증
  - 로그아웃 기능

### 2. 사용자 정보 (User Profile)
- **프로필 조회**
  - 내 프로필 조회
  - 다른 사용자 프로필 조회
  - 프로필 정보 표시 (닉네임, 소개, 프로필 이미지 등)
  
- **프로필 수정**
  - 닉네임 변경
  - 프로필 이미지 업로드/변경
  - 자기소개 작성/수정
  - 비밀번호 변경
  - 이메일 변경
  
- **프로필 통계**
  - 작성한 게시글 수
  - 작성한 댓글 수
  - 친구 수
  - 방문자 수

### 3. 게시글 관리 (Post Management)
- **게시글 조회**
  - 게시글 목록 조회 (페이징 처리)
  - 게시글 상세 조회
  - 조회수 카운팅
  - 이전/다음 게시글 네비게이션

- **게시글 작성**
  - 제목, 본문 내용 작성
  - 마크다운 형식 작성
  - 태그(Tag) 설정
  - 카테고리(Category) 분류
  
- **게시글 수정**
  - 작성한 게시글 수정
  - 수정 이력 관리
  
- **게시글 삭제**
  - 게시글 삭제 (소프트 삭제 또는 완전 삭제)

### 4. 댓글 시스템 (Comment System)
- **댓글 작성**
  - 게시글에 댓글 작성
  - 댓글 수정/삭제
  - 대댓글(Reply) 기능
  
- **댓글 조회**
  - 댓글 목록 표시
  - 댓글 페이징 처리

### 5. 검색 기능 (Search)
- **게시글 검색**
  - 제목/내용 기반 검색
  - 태그 기반 검색
  - 카테고리별 검색
  - 검색 결과 정렬

### 6. 카테고리 및 태그 관리 (Category & Tag)
- **카테고리 관리**
  - 카테고리 생성/수정/삭제
  - 카테고리별 게시글 필터링
  
- **태그 관리**
  - 태그 자동 생성 및 관리
  - 태그 클라우드 표시
  - 태그별 게시글 필터링

### 7. 친구 관리 (Friend Management)
- **친구 추가/삭제**
  - 친구 요청 보내기
  - 친구 요청 수락/거절
  - 친구 목록 조회
  - 친구 삭제 기능
  
- **친구 프로필 조회**
  - 친구의 블로그 방문
  - 친구의 게시글 조회
  - 친구 상태 확인

- **채팅 기능**
  - 실시간 1:1 채팅
  - 채팅방 목록 조회
  - 메시지 읽음/안읽음 표시
  - 채팅 내역 저장 및 조회
  - 파일/이미지 전송 기능

### 8. 아카이브 기능 (Archive)
- **시간별 아카이브**
  - 연도별 게시글 목록
  - 월별 게시글 목록
  - 날짜별 게시글 목록

### 9. 관리자 기능 (Admin Features)
- **대시보드**
  - 전체 통계 조회 (게시글 수, 댓글 수, 방문자 수 등)
  - 최근 활동 내역
  
- **시스템 설정**
  - 블로그 제목, 설명 등 기본 정보 설정
  - 테마 설정
  - 공개/비공개 설정
